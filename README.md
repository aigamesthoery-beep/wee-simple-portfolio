# Suppharoek Wee | Data Analyst Portfolio

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript. This site is designed to showcase data analysis projects and professional experience with a clean, developer-focused aesthetic.

## 🚀 Features

- **Modern Dark Theme:** Default dark mode with a deep slate/blue color palette.
- **Light/Dark Toggle:** A dynamic theme switcher with persistent settings (stored in `localStorage`).
- **Responsive Design:** Optimized for both desktop and mobile viewing.
- **Project Grid:** Interactive cards for showcasing data analysis projects with hover effects.
- **Smooth Animations:** Scroll-triggered animations and smooth navigation.
- **Custom Typography:** Uses "JetBrains Mono" for a clean, technical look.

## 🛠️ Built With

- **HTML5:** Semantic structure.
- **CSS3:** Custom properties (variables), Flexbox, and CSS Grid.
- **JavaScript (ES6):** Theme toggling logic and Intersection Observer API for animations.
- **Google Fonts:** Integration for professional typography.

## 📂 Project Structure

```text
docs/
├── index.html    # Main structure and content
├── style.css     # Theming and layout styles
└── script.js    # Interactivity and theme logic
```

## 🎨 Customization

To personalize this portfolio:
1.  **Content:** Update the `About Me` and `Projects` sections in `index.html`.
2.  **Styles:** Adjust the colors in the `:root` and `[data-theme="light"]` sections of `style.css`.
3.  **Links:** Replace the placeholder social links in the footer.
